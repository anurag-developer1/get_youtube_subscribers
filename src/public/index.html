Get YouTube Subscribers API

About

Welcome to the Get YouTube Subscribers API. This application allows you to retrieve information about YouTube subscribers using a MongoDB database.

Available Endpoints

  • GET all subscribers: /subscribers

    Returns an array of all subscribers.

  • GET subscribers' names and channels: /subscribers/names

    Returns an array of subscribers with only name and subscribedChannel fields.

  • GET subscriber by ID:

    /subscribers/:id

    Returns a single subscriber object with the given ID. If the ID doesn't exist, it returns a 400 error.

    Note: Visit /subscribers and copy the subscriber ID from any user and paste it in the input box.

Usage

To use this API, click on the endpoint URLs above or use the input field and button for the ID-specific endpoint. The browser will send a GET request to the specified URL.
